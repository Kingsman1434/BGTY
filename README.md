# BGTY — Парсер расписания БГТУ

Автоматический парсер расписания учебных занятий с официального сайта Брянского государственного технического университета.

### Функционал
- Выбор семестра, факультета, уровня образования и группы
- Поддержка чётной/нечётной недели
- Извлечение данных из HTML-таблиц
- Устойчивость к изменениям на сайте (explicit waits)
- Вывод структурированного расписания

### Технологии
- Python
- Selenium + WebDriver Manager
- BeautifulSoup (если используешь)
- Logging

### Как использовать
```bash
pip install -r requirements.txt
python BGTY.py
