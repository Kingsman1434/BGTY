# BGTY — University Schedule Parser 📅

Автоматизированный инструмент для сбора и структурирования расписания занятий с официального сайта Брянского государственного технического университета (БГТУ).

### 🌟 Ключевые функции
* **Интеллектуальная навигация:** Автоматический выбор семестра, факультета и группы через эмуляцию действий пользователя.
* **Учет четности недель:** Логика обработки расписания для четных и нечетных недель.
* **Headless Mode:** Работа браузера в фоновом режиме для экономии ресурсов сервера/ПК.
* **Устойчивость к изменениям:** Использование `explicit waits` (явных ожиданий) вместо жестких задержек (`sleep`), что делает парсер стабильным при медленном интернете.

### 🛠 Стек технологий
* **Python 3.10+**
* **Selenium WebDriver:** Основной движок для взаимодействия с DOM.
* **WebDriver Manager:** Автоматическая установка и обновление драйверов браузера.
* **Logging:** Система логирования этапов работы и ошибок.

### 📋 Как это работает
Скрипт имитирует путь студента на сайте:
1. Запуск браузера в режиме `headless`.
2. Переход в раздел расписания.
3. Последовательный выбор параметров из выпадающих списков.
4. Парсинг итоговой HTML-таблицы и вывод структурированных данных в консоль (или сохранение).
